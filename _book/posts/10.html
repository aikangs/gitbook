在MySQL字段中使用逗号分隔符

在MySQL字段中使用逗号分隔符

    mysql> SELECT * FROM content;
+----+------+
| id | tags |
+----+------+
|  1 | 1,2  |
|  2 | 2,3  |
+----+------+
2 rows in set (0.01 sec)

不需要使用like来查询


SELECT * FROM content WHERE FIND_IN_SET('2', tags) AND id <> 1
